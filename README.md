# Gemini CLI

A command-line interface for interacting with the Gemini API.

## Features

*   **Chat with Gemini:** Have interactive conversations with the Gemini model.
*   **Run commands:** Execute shell commands and get Gemini's help with the output.
*   **File system operations:** Read, write, and list files on your local machine.
*   **Web search:** Search the web with Google Search.

## Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/SyedaMehakShah/geminicli.git
    ```
2.  Install the dependencies:
    ```bash
    pip install -r requirements.txt
    ```

## Usage

```bash
python cli.py
```
